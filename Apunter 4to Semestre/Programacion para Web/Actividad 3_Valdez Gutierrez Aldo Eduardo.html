Aqui se muestra el uso de las etiquetas de formateo de texto en html

En este documento se muestra el uso de las etiquetas de formateo de texto en html, como lo son:

Etiquetas de negritas

Las etiquetas de negritas se utilizan para darle un aspecto de negritas una palabra o frase en un texto, estas etiquetas son 2:

  • <strong> Esta primer etiqueta es para negritas con mayor importancia semantica
  • <b> Esta etiqueta funciona como la anterior pero se usa para cualquier texto

Etiquetas de cursivas

Las etiquetas de cursivas se utilizan para dar el estilo de cursiva al texto, estas etiquetas son 2:

  • <em> Esta primer etiqueta es para cursiiva que tienen mayor importancia semantica
  • <i> Esta etiqueta funciona como la anterior pero se usa para cualquier texto

Etiqueta de marcatextos

Las etiquetas de marcatextos se utilizan para resaltar el texto, la etiqueta es:

  • <mark> Ejemplo de como queda como si fuera un marcatextos

Etiqueta Small (Chico)

La etiqueta small se utiliza para darle un tamaño mas pequeño al texto, la etiqueta es:

  • <small> Ejemplo de la etiqueta Small

Etiqueta de texto eliminado

La etiqueta de texto eliminado se usa darle un aspecto de eliminado al texto, la etiqueta es:

  • <del> Ejemplo de la etiqueta de texto eliminado

Etiqueta de texto Insertado

La etiqueta de texto insertado se usa para darle un aspecto de insertado al texto, la etiqueta :

  • <ins> Ejemplo de la etiqueta de texto insertado

Etiqueta de Subindice

La etiqueta de subindice se usa para darle un aspecto de subindice al texto, la etiqueta es:

  • I<sub> Ejemplo de la etiqueta de subindice

Etiqueta de Superindice

La etiqueta de superindice se usa para darle un aspecto de superindice al texto, la etiqueta es:

  • I<sup> Ejemplo de la etiqueta de superindice
